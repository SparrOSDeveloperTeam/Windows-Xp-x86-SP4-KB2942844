    • Welcome


    Welcome to the Windows Xp Service Pack 4 Unofficial Cd installation. Click the above button to install the KB2942844 update.






    Changes


    The following changes will be made to the system:




    • Windows Imaging Component (WIC) will be installed.
      .Net Framework (KB2468871) will be updated to 4.0.
      Windows Installer (KB942288) will be updated to 4.5.
      Windows Security Update (KB2761226) will be applied.
      Windows Security Update (KB2838727) will be applied.
      Windows Security Update (KB28392299) will be applied.
      Windows Security Update (KB29143678) will be applied.
      Windows Security Update (KB25362761) will be applied.
      CSDVersion will be updated to 400.
      Installed will be updated to 00000001.
    •




    System Requirements


    The following must comply before applying said update:




    • 1024MB Ram or more.
      Windows Xp x86 Home, Professional, TabletPC, MediaCTR, Starter, or K/KN must be installed.
      3GB available space.
      Service Pack 3 must be installed.
